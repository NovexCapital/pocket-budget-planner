# Pocket Budget Planner

Mobile-first React PWA budget planner.

## Deploy on Vercel

1. Go to Vercel.
2. Import this GitHub repository.
3. Framework preset: Vite.
4. Build command: `npm run build`
5. Output directory: `dist`
6. Deploy.

## Install on Android

1. Open the deployed Vercel link in Chrome.
2. Tap the three-dot menu.
3. Tap **Add to Home screen** or **Install app**.
4. Open it from your home screen like a normal app.

## Notes

The current version stores data locally in the browser using `localStorage`.
